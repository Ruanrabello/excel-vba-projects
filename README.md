# Excel VBA Projects

Este repositório contém projetos desenvolvidos em **VBA (Visual Basic for Applications)** com foco em **automação no Excel**, otimização de tarefas e manipulação de dados.

## 📌 Objetivo

O objetivo deste repositório é reunir exemplos práticos de automação utilizando **VBA**, demonstrando soluções para problemas comuns no uso do Excel no ambiente profissional.

## 🚀 Tecnologias utilizadas

- Microsoft Excel
- VBA (Visual Basic for Applications)

## 📂 Conteúdo

Neste repositório você encontrará:

- Macros para automação de tarefas
- Scripts para manipulação de dados
- Exemplos de automação em planilhas
- Projetos práticos utilizando VBA

## 💡 Exemplos de aplicações

- Automatização de relatórios
- Tratamento e organização de dados
- Criação de botões e formulários no Excel
- Otimização de processos repetitivos

## 📈 Objetivo profissional

Este repositório faz parte do meu processo de aprendizado e desenvolvimento na área de **tecnologia e análise de dados**, com foco em **automação e produtividade utilizando Excel e VBA**.
